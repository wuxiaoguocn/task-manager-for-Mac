# 性能监视器

一个适用于 macOS 的系统性能监视工具，功能类似于 RunCat，但更加轻量和美观。

## 功能特点

### 📊 系统监控
- **CPU 使用率** - 实时显示系统 CPU 占用，含系统/用户/闲置三项数据
- **内存使用率** - 监控内存占用、压力、App 内存和联动内存
- **存储容量** - 显示磁盘已用/总计空间
- **电池状态** - 电量百分比、充电状态、循环次数、温度、电源功率
- **网络连接** - 实时上传/下载速度、IP 地址

### 🎨 界面设计
- 液态玻璃（Glassmorphism）视觉效果
- 毛玻璃背景（VisualEffectView）
- 自适应图标（乌龟/兔子/火焰/警告）
- 菜单栏实时显示 CPU 状态

### ⚙️ 可自定义设置
- 显示/隐藏菜单栏图标和 CPU 百分比
- 数据刷新间隔（1s / 2s / 3s / 5s / 10s）
- 温度单位（摄氏度 / 华氏度）
- 登录时自动启动
- CPU / 内存警告阈值（50%~95%）
- 警告通知间隔（0~60 分钟）
- 独立开关各监控模块（内存/存储/电池/网络）

### 🔬 实验性功能
> ⚠️ 需要输入验证码解锁

解锁后可在菜单栏额外显示：
- 内存占用率
- 磁盘占用率
- 电池电量
- 网络上传/下载速度

## 系统要求

- macOS 15.6 或更高版本
- Apple Silicon 或 Intel Mac

## 安装

1. 克隆项目
2. 使用 Xcode 打开 `性能监视器.xcodeproj`
3. 选择目标设备为 `My Mac`
4. 点击运行（⌘R）

## 使用说明

- 点击菜单栏图标打开主界面
- 主界面底部工具栏可快速打开活动监视器、设置和退出
- 设置窗口可通过主界面底部"设置"按钮打开
- 实验性功能在设置 → 实验性功能 标签页中解锁

## 技术栈

- SwiftUI
- IOKit（电池/硬件信息）
- Mach API（CPU/内存）
- UserNotifications（系统通知）
- ServiceManagement（登录启动）
